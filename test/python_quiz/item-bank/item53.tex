\question
Is the following Python code valid?

\begin{lstlisting}
try:
    # Do something
except:
    # Do something
finally:
    # Do something
\end{lstlisting}
\begin{enumerate}
\item no, there is no such thing as finally
\item no, finally cannot be used with except
\item no, finally must come before except
\item yes
\end{enumerate}
